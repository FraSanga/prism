# Prism Editor PRO

A graphical editor for creating and simulating optical configurations with prisms.

## Installation

1. Create a virtual environment:
```bash
python -m venv venv
```

2. Activate the environment:
* Windows:
```bash
venv\Scripts\activate
```

* macOS/Linux:
```bash
source venv/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

![Editor Screenshot](images/editor.png)

```bash
python editor.py
```

For detailed instructions on how to use the editor, please refer to the [MANUAL.md](MANUAL.md) file.
